    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="user-content-item-based-collaborative-filtering-c-implementation" class="anchor" href="#item-based-collaborative-filtering-c-implementation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Item-based Collaborative Filtering C++ Implementation</h1>

<p>The original implementation can be found at: </p>

<p><a href="http://www.yekeren.cn/blog/archives/1036">http://www.yekeren.cn/blog/archives/1036</a></p>

<p>This modified version implements a Correlation Similarity function. The function can be modifed in "itemcf.cpp" to other customized Similarity Function.</p>

<p>How-to-use:
Modify "main.cpp" to include the dataset.</p>

<p>One testing dataset is provided from MovieLen 1M rating: </p>

<p><a href="http://grouplens.org/datasets/movielens/">http://grouplens.org/datasets/movielens/</a>.</p>

<p>Please include references when using any of the implementation or dataset.</p></article>
